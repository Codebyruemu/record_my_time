# record_my_time
simple utility app that helps you keep track of time invested in a task. you can pause if distracted with other activities and play when back to task. however,the task session time auto-stop at bechmarked time threshold.
this simple project was personally designed to keep me focus on study for a period of 3hrs (the threashold time). feel free to adjust the time in utilityfuc.js file.
